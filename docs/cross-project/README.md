# Cross-Project-Todos (Firefly ↔ Wayfinder)

Firefly und Wayfinder sind getrennte Repos mit getrennten Claude-Sitzungen,
gekoppelt nur über den **CAT062-Draht-Vertrag** (siehe `../ICD-CAT062.md`).
Schnittstellen-relevante Themen werden als GitHub-Issues im jeweils anderen
Repo geführt (Label `from-firefly` bzw. `from-wayfinder`) und hier
referenziert. Details: `CLAUDE.md` Abschnitt 9.

- **`todo-for-wayfinder.md`** — Themen aus Firefly, die Wayfinder betreffen.
- **`todo-for-firefly.md`** — Themen aus Wayfinder, die Firefly betreffen.
